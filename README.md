# Hi! Fly Guy 高频词课件

英语启蒙课件 —— 围绕绘本《Hi! Fly Guy》(Tedd Arnold) 中出现的**高频词（Sight Words 视读词）**设计。

- **适用对象**：小学 1–4 年级
- **学习目标**：认读、理解并能拼写 32 个高频词；做到「见词即读」
- **制作单位**：小姨英语教育

## 二、互动课件（courseware.html）

六个标签页：

1. **📖 学习单** —— 32 个词的卡片墙，点卡片听例句、点喇叭听单词，可「全部朗读」。
2. **🃏 单词闪卡** —— 看词先读，点击翻面看中文 + 例句，可打乱顺序。
3. **🔗 配对连线** —— 英文 ↔ 中文配对，配对成功变绿。
4. **👂 听音选词** —— 朗读单词，四选一，自动计分。
5. **✏️ 句子填空** —— 读句子选出空格里的词，自动计分。
6. **🔤 拼写单词** —— 看中文 + 听发音，点字母把单词拼出来。

**发音功能**：使用浏览器内置语音合成（Web Speech API）。
- 推荐用 **Chrome / Edge** 打开效果最佳；
- 若没声音，请检查系统是否安装英文语音包、音量是否打开。

## 三、纸质学习单（worksheet.html）

A4 两页，黑白打印友好，包含：

1. **高频词总表**（认读 + 描写）
2. **描红练习**（描一遍 + 自己写一遍）
3. **连一连**（英文—中文）
4. **选词填空**（含词库）
5. **看中文写单词**

**打印方法**：打开 `worksheet.html` → 右上角点「🖨 打印 / 存为 PDF」。
- 顶部操作栏和参考答案在打印时会**自动隐藏**（学生版）；
- 想要**教师版（带答案）**：先点「👁 显示/隐藏答案」显示答案，再打印。
- 每次刷新页面，连线/填空/写单词的**题目会重新随机抽取**，可生成多套练习。

## 四、词表（32 个高频词）

按 Dolch / Fry 视读词表分级：

- **Pre-K**：a, the, to, for, can, not, up, down, look, my, you, in, is, away
- **K**：he, are, but, this, was, went, saw, what, with, said
- **1st**：his, know, home, fly
- **2nd**：around
- **名词**：boy, pet, name
